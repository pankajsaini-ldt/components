Tooltip Placement

Tooltip allows you to specify where it should appear in relation to the target element. It accepts the following values:

  • auto
  • auto-end
  • top-start
  • top
  • top-end
  • right-start
  • right
  • right-end
  • bottom-end
  • bottom
  • bottom-start
  • left-end
  • left
  • left-start
<Tooltip content="I'm on top" placement="top">
  {(tooltipProps) => (
    <Button {...tooltipProps}>
      Top
    </Button>
  )}
</Tooltip>
<Tooltip content="I'm on bottom" placement="bottom">
  {(tooltipProps) => (
    <Button {...tooltipProps}>
      Bottom
    </Button>
  )}
</Tooltip>
<Tooltip content="I'm on the left" placement="left">
  {(tooltipProps) => (
    <Button {...tooltipProps}>
      Left
    </Button>
  )}
</Tooltip>
<Tooltip content="I'm on the right" placement="right">
  {(tooltipProps) => (
    <Button {...tooltipProps}>
      Right
    </Button>
  )}
</Tooltip>
